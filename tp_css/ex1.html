Exercice 1 - Réponses

1. Sélecteurs :

  • Tous les liens → .main-nav a
  • Lien actif → .main-nav a.active
  • Article featured → .post.featured
  • Premier paragraphe → article p:first-of-type
  • li pairs → li:nth-child(even)

2. Spécificité :

Ordre croissant :

p → .text → p.text → #main → #main .text p

Si même spécificité → la dernière règle gagne

Éviter !important → rend le code difficile à maintenir
